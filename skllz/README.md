# skllz

This directory contains the sandbox-wide skill synthesis deliverables:

- `AGENT_SKILL_LESSONS_LIST.md`
  - Raw extraction of useful agent-building lessons by reviewed directory.
- `AGENT_SKILL_LESSONS_GROUPED.md`
  - Same material reorganized by skill type, with candidate skill names.
- `00_project_inventory.md`
  - Generated inventory of project structures and key files.
- `05_directory_briefs.md`
  - Heading-level brief per top-level directory.
