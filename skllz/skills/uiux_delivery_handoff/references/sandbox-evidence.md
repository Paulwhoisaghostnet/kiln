## Lessons Applied
1. Architecture audits can uncover hidden renderer/state coupling failures that block UX correctness.
2. Explicit workflow redesign (with manual gates where needed) improves reliability and user trust.
3. Handoff quality depends on concrete run/setup docs, env expectations, and operational next actions.
4. Phased remediation plans outperform vague backlog notes when systems are brittle.
