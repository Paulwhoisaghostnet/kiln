# Subagent Feedback Schema

Each agent should provide:
- `README.md`
- `SKILL_FEEDBACK.md`
- `DOC_PACK/` with topic-specific docs

`SKILL_FEEDBACK.md` minimum sections:
- What helped
- What was missing
- Improvement proposals

Recommended extraction fields:
- top 5 knowledge nuggets
- explicit failure/fallback patterns
- deterministic QA hooks used
