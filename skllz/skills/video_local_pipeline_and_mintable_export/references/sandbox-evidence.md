## Lessons Applied
1. Local ffmpeg-based pipelines are reliable for segmentation, preview, and export.
2. Mint-ready outputs must be self-contained with no external dependency chain.
3. Installation and toolchain verification reduce avoidable pipeline failures.
4. Hosted WASM flows require explicit MIME/header/deploy configuration.
5. Export QA should validate compiled artifacts directly and capture console health evidence.
