# Artifact Matrix Template

For each debug pass, capture:

- `output/.../shot-0.png` (baseline frame)
- `output/.../shot-1.png` (post-fix comparable frame)
- `output/.../state-0.json` (baseline state)
- `output/.../state-1.json` (post-fix state)
- Optional additional long-run checkpoints (`+5s`, `+10s`)

Also record:
- Seed/mode used
- Camera mode used
- Any known automation limitations
