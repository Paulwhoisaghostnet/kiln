# Chain Signal Game Loop Patterns

## Loop Pattern
1. Read bounded chain/indexer window.
2. Normalize into stable signal vector.
3. Map signals to mechanics through bounded curves.
4. Smooth transitions over time.
5. Export deterministic debug state for QA.

## Example Signal Families
- Activity volume -> hazard pacing.
- Metadata trait diversity -> ability recovery options.
- Entrypoint mix -> biome/track mood modifiers.

## Safety Rules
- Keep player agency floor (minimum speed, minimum recovery chance).
- Cap volatility influence to avoid unwinnable spikes.
- Separate progression systems from external data shocks.
